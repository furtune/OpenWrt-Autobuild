#### 特色及功能

- 支持 ECC 密钥
- 已进入上游稳定分支的设备，后续只跟进稳定分支
- 默认密码: 无
- 部分官方源存在，依赖性不强的应用不考虑集成，有依赖问题再考虑集成进固件
- 弱鸡的 cpu，不考虑集成 golang 应用
